# Nirnama: Rentak Pendekar (Projek MVP)

Permainan ini dibina khas sebagai **Minimum Viable Product (MVP)** / prototaip konsep untuk dipersembahkan kepada pemilik asal harta intelek (IP) **Nirnama**. 

### Status Projek:
* **Bukan Permainan Rasmi**: Projek ini tidak mempunyai kaitan rasmi dengan pelancaran komersial atau pengeluaran rasmi pihak pemilik IP Nirnama pada waktu ini.
* **Tujuan Pembentangan**: Dibina semata-mata untuk menunjukkan potensi konsep mekanik permainan perlawanan 2D berasaskan watak-watak daripada dunia Nirnama kepada pihak pencipta dan pemilik asal.
* **Prototaip Demo**: Segala reka bentuk, grafik, dan mekanik permainan di dalam demo ini adalah sekadar demonstrasi teknikal awal untuk tujuan persembahan idea sahaja.

## Roster Pendekar & Kuasa Khas (Special Moves)

Berikut adalah senarai pendekar yang boleh dipilih berserta serangan istimewa mereka:

1. **Nirnama** (Balanced / Shoto)
   * **Kuasa Khas**: *Pukulan Angin (Hadouken Wave)*
   * **Butang Combo**: `S ➔ S+D ➔ D + F` (Bawah, Bawah-Kanan, Kanan + Punch) atau `R` / `Butang K` (pada peranti mudah alih)
   * **Penerangan**: Melancarkan aura tenaga berbentuk keris sakti berwarna merah merentasi medan tempur untuk menyerang dari jarak jauh.

2. **Cendana** (Zoner / Mid-Range)
   * **Kuasa Khas**: *Tether Strike (Scarf Sweep)*
   * **Butang Combo**: `S ➔ S+A ➔ A + F` (Bawah, Bawah-Kiri, Kiri + Punch) atau `R` / `Butang K` (pada peranti mudah alih)
   * **Penerangan**: Menyerang menggunakan cemati/selendang sakti jarak jauh bercahaya turquoise untuk melibas dan menolak musuh ke belakang.

3. **Nadim** (Rushdown / Agile)
   * **Kuasa Khas**: *Larian Kuning (Dash Charge)*
   * **Butang Combo**: Tahan `A` (1 saat) ➔ `D + F` (Tahan Kiri, lepas tu Kanan + Punch) atau `R` / `Butang K` (pada peranti mudah alih)
   * **Penerangan**: Larian pantas yang diselubungi aura cahaya kuning tebal serta bayangan laju untuk merempuh dan memecahkan benteng pertahanan musuh (*block-breaker*).

4. **Sang Syaitan** (Heavy Grappler / Boss)
   * **Kuasa Khas**: *Genggaman Maut (Command Grab)*
   * **Butang Combo**: `S ➔ S + G` (Bawah, Bawah + Kick) atau `R` / `Butang K` (pada peranti mudah alih)
   * **Penerangan**: Menggunakan cengkaman jarak dekat yang tidak boleh dihadang (*unblockable*) untuk mengangkat dan menghempas musuh secara kejam.
